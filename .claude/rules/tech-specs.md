## Technical Specifications

- **CSS framework** — Tailwind CSS via CDN is mandatory:
  ```html
  <script src="https://cdn.tailwindcss.com"></script>
  ```
- **Images** — When source images are not provided, use placeholders from `https://placehold.co/`
- **Responsive** — Mobile-first design paradigm by default
- **Output** — Single `index.html` file unless a multi-file structure is explicitly requested by the user
